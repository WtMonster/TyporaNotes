### 第一章

#### double理解补充

##### 原理

| 符号位（S）：1bit | 指数位（E）：11bit | 尾数位（M）：52bit |      |
| ----------------- | ------------------ | ------------------ | ---- |

所以指数位取值

指数取值的范围是-1023~+1024

2^1024=10^308，以此来得到double值取值范围

##### 使用

| 变量名            | 作用                     |
| ----------------- | ------------------------ |
| MAX_VALUE         | 返回最大双精度数         |
| MIN_VALUE         | 返回最小双精度数         |
| NaN               | 表示非数值类型的双精度数 |
| NEGATIVE_INFINITY | 返回负无穷双精度数       |
| POSITIVE_INFINITY | 返回正无穷双精度数       |
| TYPE              | 返回当前类型             |
