# idea快捷键

| 作用                     |                   |
| ------------------------ | ----------------- |
| 生成接口的实现类         | alt+enter         |
| 选择要继承的方法         | ctrl + o          |
| 生成测试类               | ctrl + shift + t  |
| 新建指定文件             | alt+insert        |
| 用行注释注解多行         | ctrl+/            |
| 用块注释注解多行         | ctrl+shift+/      |
| 复制并粘贴光标当前所在行 | ctrl + d          |
| 快速删除光标所在行       | ctrl + y          |
| 执行某个插件             | ctrl+ctrl         |
| 翻译插件                 | ctrl+shift+y      |
| 将当前行上移下移         | alt+shift+↑或者+↓ |
| 空出光标所在行下一行     | ctrl+shift+enter  |
| 清空无效import           | ctrl+alt+o        |
| 查看接口实现类           | ctrl+h            |
| 搜索类                   | shift+shift       |
| 代码格式化               | ctrl+alt+l        |
| 打开指定文件             | ctrl+shift+n      |
| 全选                     | ctrl+a            |
|                          |                   |
